    The goal is to create the best experiences for people

    Project Background

    My portfolio is a personal showcase presenting my skills in UI/UX design, front-end development, and my previous projects. The goal was to create a functional, aesthetic, and intuitive website that not only presents my projects but also my technical skills and design approach.

    Context

    This project was a personal initiative to create an online professional portfolio. The focus was on building an optimized user interface with a responsive design. The website is built using HTML, CSS, JavaScript, and employs modern front-end development techniques.

    Process

    • Defining Goals: The main objective was to build an online professional portfolio.
    • Wireframes and Mockups: Created wireframes and mockups on Figma to establish the site’s structure.
    • Front-End Development: Used HTML, CSS, and JavaScript to develop the main page and individual project pages.
    • User Testing: Conducted usability tests to ensure smooth navigation and design consistency across various screen resolutions.
    • Feedback and Iterations: Continuous improvements after each user test to refine features and interface.
    • Mobile Optimization: The design is fully responsive, optimized for mobile and tablet devices.

    Research

    As part of creating my portfolio, I conducted an in-depth study of portfolios from established professionals online. This research helped me understand the expectations of recruiters and my target audience. The key takeaways I gathered include:

    • Simplicity and Minimalism: Many professionals favor a simple and clean design, where the essentials are highlighted without distractions.
    • Clear Presentation of Projects: Each project is accompanied by a concise explanation of its context, the process followed, and the results achieved.
    • Personalization: Many portfolios incorporate personalized elements that reflect the creator's personality and style while remaining professional.
    • Accessibility and Navigation: Intuitive navigation and a responsive design are essential to ensure a good user experience across all devices.

    These findings were incorporated into the design of my own portfolio to ensure it is not only aesthetic but also functional and aligned with industry expectations.

    Design Goals

    • Create a responsive website that adapts to all screen sizes.
    • Present projects and skills clearly and concisely.
    • Optimize the user experience with smooth navigation and a simple interface.
    • Integrate both dark and light modes for a personalized experience.
    • Highlight the creator's personality and unique style through design.

    Design Problems

    • Finding a balance between aesthetic design and a functional interface.
    • Ensuring a smooth user experience despite the use of multiple technologies.
    • Choosing the right colors and typography to ensure readability and a professional atmosphere.

    Design Decisions

    • Used a minimalist design with interactive elements to maintain focus on the projects.
    • Implemented a simple navigation system with a sidebar for quick access to different sections.
    • Integrated a dark and light theme toggle button to enhance the user experience.
    • IChose a modern, readable typeface to facilitate reading on all devices.

    Reflections

    This project allowed me to apply everything I’ve learned in UI/UX design and web development. I understood the importance of iteration in the design process and the impact of a smooth user experience on the perception of the product. Reflecting on my own portfolio, I realized the importance of visual coherence and simplicity in presenting information. For future versions, I would like to explore more advanced interactive features and improve the site's accessibility.
